# Kizuna

Kizunaは、GitHub Issuesで行うプロジェクト作業と、カレンダーやリマインダーなどの個人向け生産性ツールをつなぐことを検討するソフトウェアです。Dokoniエコシステムの独立したプロジェクトとして開発しています。

## 目的

プロジェクトの情報を日々の作業の中で無理なく活用できる、保守しやすい連携方法を探ります。新しい機能を一つずつ理解し検証できるよう、開発は小さなMockに分けて進めます。

## 開発の進め方

- Mock 1とMock 2は完了しています。
- Mock 2でモジュール化された基盤を整えました。
- Mock 3ではその構成を引き継ぎ、GitHub Issuesの作成と更新に取り組みます。
- 変更は段階的に行い、テストを重視します。

Kizunaは仕様ではなくソフトウェアです。AMTSをプロジェクト知識の整理に利用することはありますが、両者は独立したプロジェクトです。
