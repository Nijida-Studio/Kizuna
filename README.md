# Kizuna

Kizuna ist ein Open-Source-Softwareprojekt im Dokoni-Ökosystem. Es untersucht, wie sich Arbeit in GitHub Issues mit persönlichen Produktivitätssystemen wie Kalendern und Erinnerungen verbinden lässt.

Die Entwicklung erfolgt schrittweise in klar abgegrenzten Mocks. Mock 1 und Mock 2 sind abgeschlossen; Mock 3 erweitert die bestehende modulare Architektur um das Erstellen und spätere Aktualisieren von GitHub Issues.
