Kizuna explores how project work in GitHub Issues can connect with personal productivity systems such as calendars and reminders. It is an independent software project within the Dokoni ecosystem.

> Kizuna is software, not a specification. Supported systems and release scope are not yet committed at this early stage.

Mock 1 and Mock 2 are complete. Mock 3 builds on the modular Mock 2 architecture and focuses on creating and later updating GitHub Issues.
